## 問題文
0 以上 
255 以下の整数 
A,B が与えられます。  
A xor C=B となる 
0 以上の整数 
C を求めてください。  

なお、そのような 
C はただ 
1 つ存在し、
0 以上 
255 以下であることが証明されます。
## 入力
A B
## 出力
答えを出力せよ。
## 入力例
3 6
## 出力例
5  
3 は 二進表記で 
11、
5 は二進表記で 
101 なので、これらの 
xor は二進表記で 
110 であり、十進表記で 
6 です。  

このように、
3 xor 5=6 となるので、答えは 
5 です。
## 解法

```python
a,b=map(int,input().split())

for i in range(256):
    if a^i==b:
        print(i)
