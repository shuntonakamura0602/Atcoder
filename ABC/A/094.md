## 問題文
猫と犬が合わせて A+B 匹います．  
このうち A 匹は猫であることがわかっていますが，残りの B 匹は猫と犬のどちらであるかわかっていません．  
この A+B 匹の中に，猫がちょうど X 匹いるということはありうるかどうか判定してください．
## 入力
A B X
## 出力
猫がちょうど 
X 匹いるということがありうるならば YES を，ありえないならば NO を出力せよ．
## 入力例
3 5 4
## 出力例
YES
## 解法

```python
A, B, X = map(int, input().split())
if A <= X <= A+B:
    print('YES')
else:
    print('NO')
