## 問題文
3 本の柱が等間隔に並んでいます。  
柱の高さは左から順に a メートル, b メートル, c メートル です。  
柱の先端が同一直線上に並んでいる時、つまりb−a=c−b を満たしているとき、 この柱の並び方を美しいと呼びます。  
柱の並び方が美しいかどうかを判定してください。
## 入力
a b c
## 出力
柱の並び方が美しい場合 YES を、そうでない場合 NO を 1 行に出力せよ。
## 入力例
2 4 6
## 出力例
YES
## 解法

```python
a,b,c = map(int,input().split())

x = b - a
t = c - b

if x == t:
    print("YES")
else:
    print("NO")
