## 問題文
高橋君は、プログラミングコンテスト AXC001 に参加しており、問題 A にコードを提出しました。  
この問題には 
N 個のテストケースがあり、すべてのテストケースに正解した場合のみ提出は AC となります。  
高橋君の提出は、
N 個のテストケースのうち 
M 個のテストケースに正解しました。  
高橋君の提出が AC となるか判定してください。
## 入力
N M
## 出力
高橋君の提出が AC となる場合は Yes, そうでない場合は No と出力せよ。
## 入力例
3 3
## 出力例
Yes
## 解法

```python
N,M = map(int,input().split())
if N == M:
  print('Yes')
else:
  print('No')
